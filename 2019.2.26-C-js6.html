一、背景介绍

AngularJS 控制器 控制 AngularJS 应用程序的数据。
AngularJS 控制器是常规的 JavaScript 对象。

二、知识剖析

AngularJS 应用程序由 ng-app 定义。应用程序在 div 内运行。
ng-controller="myCtrl" 属性是一个 AngularJS 指令。用于定义一个控制器。

在 html 中使用 ng-controller 指令实例化控制器时，会产生一个新的作用域对象。在同一个页面的
ng-app 作用域下可以通过 ng-controller 实例化多个控制器对象。但是，每个控制器对应的作用域对象只能与
ng-controller 指令所在标签的开始与结束标签之间的 DOM 元素上建立数据绑定。
