一、背景介绍

AJAX = 异步 JavaScript 和 XML。 AJAX 是一种用于创建快速动态网页的技术。 通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。 传统的网页（不使用 AJAX）如果需要更新内容，必需重载整个网页面。

二、知识剖析

Ajax 就是用 JS 发起一个请求，并得到服务器返回的内容。这跟以前的技术最大的不同点在于「页面没有刷新」，改善了用户体验，仅此而已。 那么我们如何发送一个 ajax 请求呢？ 1. 创建一个对象 XMLHttpRequest var xhr = new XMLHttpRequest(); 为了支持 ie6 以及更早的版本，要 var xhr=new ActiveXObject() 2. 监听请求成功后的状态变化 3. 设置请求参数 4. 发起请求 5. 操作 DOM，实现动态局部刷新

一个完整的的http请求，通常有下面7个步骤

  • 建立TCP连接
  • web浏览器向web服务器发送请求命令
  • web浏览器发送请求头信息
  • web服务器应答
  • web发送应答头信息
  • web服务器向浏览器发送数据
  • web服务器关闭Tcp连接

一个完整的请求由四部分组成

  • HTTP请求的方法或者是动作，比如是GET或者是POST请求
  • 正在请求的url，也就是常说的网址
  • 请求头，包含一些客户端环境信息，身份验证信息等
  • 请求体，也就是请求正文，
