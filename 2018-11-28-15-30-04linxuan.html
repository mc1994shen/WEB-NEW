1.背景介绍

什么是JS数据类型
数据类型本质上是由一组无序的名值对组成的
由于ECMAScript是不支持任何自定义类型的机制
所以所有的数值都是两大类的数据类型组成的


哪两大类的数据类型
第一个是复杂数据类型
第二个是基本数据类型


基本数据类型有哪些
Undefined,Null,Boolean,Number,String

复杂数据类型有哪些
Object



2.知识剖析

简单数据类型中的Undefined
1>.
就是如果这个数值没有定义
或者这个类型只有一个值
在使用var声明的以后没赋值给他



Boolean
如果这个数值是布尔值
只有两个类型的数值
true,false

String
如果这个数值是字符串
是由零或者最多16位的Unicode字符组成的字符序列
可以是“”或者‘’来表示都是可以的

Number
这个数值是数字
有整数和浮点数


Null
这个类型是第二个只有一个值的数据类型
特殊的值是null，null值表示一个空对象指针
所以返回数值是object

复杂数据类型中的object
2>.
如果这个值是对象或者null
就是数据和功能的集合
可以通过执行new操作符后跟要创建的
对象类型和名称来创建

3.还有其他判断数据类型的方法吗？

1.使用typeof会返回object instanceof会报错 constructor会报错 Object.prototype.toString.call返回[object Null]

2.if(typeof a!=”undefined”){alert(“ok”)}，而不要去使用 if(a) 因为如果 a 不存在（未声明）则会出错，对于 Array,Null 等特殊对象使用 typeof 一律返回 object，这正是 typeof 的局限性

3.1.typeof：检测数据类型的运算符 2.instanceof：检测某个实例是否属于这个类 3.constructor：获取当前实例的构造器 4.Object.prototype.toString.call：获取当前实例的所属类信息
